Техническая поддержка и линия консультаций по работе с расширением доступны в линии
поддержки 1С-Коннект «Сорокин и К: Управление франчайзи для УНФ» в рабочие дни с 8.00 по
16.00 (по московскому времени). Линия доступна всем пользователям в РФ, быстрый поиск линии
по [ссылке](https://sa.1c-connect.com/join/s/hxih935nhtgsi8zx3ot58yxekc).
Обращаем Ваше внимание, что поддержка и линия консультаций по работе с расширением
обеспечивается для зарегистрированных пользователей, которые официально
приобрели продукт.

При обращении на линию поддержки обязательно указать:
- информацию о релизе «1С:УНФ» и установленных расширениях из раздела
«Конфигурация» окна «О программе», которое открывается из меню в верхней правой панели
программ 1С:Предприятие 8;

![о программе](img/1cv8c_about_programm.png)

![тех информация](img/1cv8c_tech_info.png)

- последовательность действий приводящих к ошибке;
- скопировать полный текст ошибки;
- описать как, по Вашему мнению, должна правильно работать программа.

{: .note }
**Разработчик ООО «Сорокин и К» обеспечивает исправление ошибок,
обнаруженных пользователями, в выпусках новых релизов расширения, но не
гарантирует исполнения всех пожеланий и предложений по развитию функционала
расширения.**
